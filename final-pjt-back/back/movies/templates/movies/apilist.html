http://127.0.0.1:8000/api/v1/movie/title/
요청 타입 : GET
params = { 영화 제목1, 영화 제목2, 영화 제목3, ... }
반환 값 : params에 있는 영화 중 DB에 있는 영화의 정보를 반환
목적 : chatgpt에서 추천받은 영화 중 DB에 있는 영화를 찾는 용도

http://127.0.0.1:8000/api/v1/movie/alltitle/
요청 타입 : GET
params 없음
반환 값 : 모든 db의 영화의 제목과 id만 반환
목적 : 앱 시작시 전체 영화의 제목을 저장해서 검색시 해당 검색어를 포함한 영화 자동완성

http://127.0.0.1:8000/api/v1/movie/
요청 타입 : GET
params = { order_by, page }
page : required int
order_by : `vote_average` or `popularity` default = `vote_average`
반환 값 : 데이터를 정렬 조건에 따라 정렬하며 페이지당 10개의 영화를 로드한다.
목적 : 전체 영화를 조회하는 용도

http://127.0.0.1:8000/api/v1/movie/{movie_pk}
요청 타입 : GET
params = 없음
반환 값 : 해당 pk에 해당하는 영화의 정보를 반환 함
목적 : 특정 영화 정보 조회 용도

요청 타입 : PUT
body : { popularity }
반환 값 : pk에 해당하는 영화의 popularity가 수정된 정보를 반환함
목적 : 자체 평점 조회 용도

http://127.0.0.1:8000/api/v1/review/{movie_pk}
요청 타입 : GET
params = 없음
반환 값 : pk에 해당하는 영화의 리뷰 정보를 반환 함
목적 : 영화 detail페이지에서 리뷰를 표시하는 용도

요청 타입 : POST
body : {vote, content}
vote : required float
content : 공백 가능
반환 값 : 생성된 리뷰 정보
목적 : 새로운 리뷰를 생성하는 용도

http://127.0.0.1:8000/api/v1/review/detail/{review_pk}
요청 타입 : PUT
body : {vote, content}
vote : required float
content : 공백 가능
반환 값 : 수정된 리뷰 정보
목적 : 리뷰를 수정하는 용도

요청 타입 : DELETE
params = 없음
반환값 = 성공여부 반환
목적 : 리뷰를 삭제하는 용도
